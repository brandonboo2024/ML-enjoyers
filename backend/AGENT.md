# Backend Agent — Instructions

Read these files before starting:
- `SPEC.md`
- `TASKS.md`

## Scope
- You may only edit files under `backend/`.
- Do not edit shared files (`SPEC.md`, `TASKS.md`, `INTEGRATION.md`, `CHANGELOG.md`).

## Expectations
- Maintain API contract in `SPEC.md`.
- Avoid breaking frontend expectations.
- Append your updates to `CHANGELOG.md` using the required format.
