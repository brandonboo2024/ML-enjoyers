# Frontend Agent — Instructions

Read these files before starting:
- `SPEC.md`
- `TASKS.md`
- `INTEGRATION.md`

## Scope
- You may only edit files under `frontend/`.
- Do not edit shared files (`SPEC.md`, `TASKS.md`, `INTEGRATION.md`, `CHANGELOG.md`).
- If you need a cross‑cutting change, log it in `CHANGELOG.md` for Integrator.

## Expectations
- Follow the API contract in `SPEC.md`.
- Keep UI changes compatible with the existing alert schema.
- Append your updates to `CHANGELOG.md` using the required format.
